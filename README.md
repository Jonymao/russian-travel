## Путешествие по России - проектная работа для Яндекс Практикума
---
### Обзор
Проект представляет собой одностраничный сайт, с применением технологий, которые применяются для создания адаптивности
---

## Применяемые технологии
* HTML
* Flex-box
* CSS
* Методология БЭМ
* Grid
